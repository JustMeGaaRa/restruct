# Serve

The serve command is used to serve the re:struct CLI.

## Usage

```bash
restruct serve [options]
```

## Options

| Option      | Description                             |
| ----------- | --------------------------------------- |
| `--port`    | The port to serve the re:struct CLI on. |
| `--host`    | The host to serve the re:struct CLI on. |
| `--help`    | Display help information.               |
| `--version` | Display version information.            |
